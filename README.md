# Mi Portafolio - Juan Manuel Velázquez

¡Bienvenido a mi portafolio! Aquí encontrarás una muestra de mis proyectos, mis habilidades y las tecnologías que utilizo para crear soluciones eficientes y atractivas.

## 💡 Diseño
Este portafolio ha sido diseñado utilizando los principios de **Material 3 (Material You)**, con el objetivo de crear una experiencia visual limpia, moderna y accesible. Además es el estilo que más me gusta.
